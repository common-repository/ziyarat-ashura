== Description ==

En	: Ziarat ashura wordpress plugin for english language with arabic sound.

Ar	:  إضافات ووردبريس - زيارة عاشوراء - العربیه

Fa	: افزونه ی زیارت عاشورا برای وردپرس که متن عربی و ترجمه ی فارسی در آن قرار داده شده است.همچین از صدای آقای فرهمند نیز برای پخش زیارت عاشورا استفاده شده است.

Features:

*   Simple design
*	Pause texts on hover
*   Option panel for customize
*   Html5 player to play arabic ziarat ashura
*   Shortcode to use in posts & pages

== Installation ==

1. Upload `zashura` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to the plugin option page

== Frequently Asked Questions ==

= How To Use this plugin? =

En:

After installing plugin use this shortcode to show ziarat ashura :
<code>[ziarat_ashura]</code>

You can use this code to show ziarat ashura in your theme : 

<code><?php do_shortcode('[ziarat_ashura]');?></code>

Fa :  
بعد از نصب می توانید از کد زیر در پوسته استفاده کنید .

<code><?php do_shortcode('[ziarat_ashura]');?></code>

همچنین برای نمایش در پست ها و یا صفحات نیز می توانید از شورت کد زیر استفاده کنید 

<code>[ziarat_ashura]</code>
